    简介

    通用下载服务,服务器端使用aria2作为下载器

  • 

    支持列表

    ftp,http部分bt,以及其他下载地址

  • 

    使用方法

    复制下载链接,粘贴至输入框,选择"普通"选项,点击下载

  • 

    简介

    通过所给链接的javascript和html代码分析视频和音频下载路径,下载对应的音视频,功能十分强大
